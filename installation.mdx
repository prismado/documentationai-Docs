## Verzeichnisse und Schreibrechte

```bash
/var/www/html
```

| Verzeichnis    | `chown`             | Inhalt                                                   |
| -------------- | ------------------- | -------------------------------------------------------- |
| `upload`       | <kbd>www-data</kbd> | Generierte PDF-Belege, CSV-Importe, RAW Order (Callback) |
| `api/files`    | <kbd>www-data</kbd> | Assigned Users (XML), XML für M42                        |
| `_systex/data` | <kbd>www-data</kbd> | XML-Antwort von cXML-Server (Navision)                   |
| `images/temp`  | <kbd>www-data</kbd> | QR-Codes                                                 |
