```bash

```

| Verzeichnis    | Permission          | Inhalt                                                   |
| -------------- | ------------------- | -------------------------------------------------------- |
| `upload`       | <kbd>www-data</kbd> | Generierte PDF-Belege, CSV-Importe, RAW Order (Callback) |
| `api/files`    | <kbd>www-data</kbd> | B                                                        |
| `_systex/data` | <kbd>www-data</kbd> | C                                                        |
| `images/temp`  | <kbd>www-data</kbd> | D                                                        |
