### **Funktionsliste für Spezialrollen**

- CSP: Upload von Inventarfiles und PKID-Reports mit automatisierter Verarbeitung

- CSP: Teilautomatisierter E-Mail-Versand von PKID-Reports an Clouds Team

- CSP: Registrtrierungsbestätigung mit automatischem Folgeworkflow (Status-Update, XML Inventar, Registrierungsbestätigung an M42)

- SSG: Setzen von operativen Updates bei Artikeln: Pre-provisioning, Durchgeführter (Teil-)versand

- SSG: XML-Updates an Matrix 42

- SSG: Bestellung stornieren
