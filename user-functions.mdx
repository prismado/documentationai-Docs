## Funktionsliste Standardbenutzer

- Selbst-Registrierung / Login / Logout

- 2FA mit Verwaltungsmöglichkeiten Anzeigen / Deaktivieren

- Benutzerverwaltung (Admin-Funktion)

- Benutzer nach Selbstregistrierung freigeben (Admin-Funktion)

- Bestellung aus verschiedenen externen Produktekatalogen (homologierte Geräte und Voll-Katalog)

- Automatische Übermittlung in cXML-Format an Lieferant

- Verwaltung verschiedener Stammdaten (Adminfunktion)

- Verwerf-Funktion (Bestellabbruch) mit Historisierung

- Zuweisung der Bestellpositionen an zugeteilte Departemente, selektive Bestellung von Pre-provisioning bei Image-fähigen Geräten

- Passwort vergessen-Funktion

- Artikelliste bestellter Geräte

- Automatische Übermittlung der Bewegungsdaten an PowerBI-Reporting
